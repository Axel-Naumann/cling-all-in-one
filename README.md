cling-all-in-one
================

Get and build llvm, clang and cling. Can also be used for subsequent updates.
Usage:

```
git clone https://github.com/Axel-Naumann/cling-all-in-one.git
cd cling-all-in-one
./clone.sh
```

Contact cling-dev@cern.ch if you have questions.
